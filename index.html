Data Virtual Machines: Simplifying data sharing, exploration & querying in big data environments

Co-authored, academic paper published on 2022 IEEE International Conference on Big Data; Osaka, Japan

Publication presents a graph-based conceptual model, the Data Virtual Machine (DVM) representing data of an organization. A DVM can be built quickly and agilely, using visual interfaces for schema and query management. Dataframes over DVMs are expressed visually via a simple and intuitive tool, allowing, non-IT experts can be involved in dataframing.
